Hexo博客基础

发表于 2021-01-28 更新于 2021-01-29 分类于 Hexo

前置步骤

  1. 下载并安装Node.js，安装完成后包含两个组件（Node.js和npm），在命令提示符窗口执行命令node -v和npm -v查看是否安装成功。

  2. （可选步骤）在命令提示符窗口执行命令npm install -g cnpm --registry=https://registry.npm.taobao.org，该步骤目的为将镜像源更换为淘宝镜像源以加快下载速度。使用命令cnpm -v看是否安装成功。

  3. 在命令提示符窗口执行命令cnpm install hexo-deployer-git --save安装Hexo博客部署插件。
