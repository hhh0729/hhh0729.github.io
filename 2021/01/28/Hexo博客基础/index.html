    Hexo博客基础

    发表于 2021-01-28 更新于 2021-01-29 分类于 Hexo

    前置步骤

    1. 下载并安装Node.js，安装完成后包含两个组件（Node.js和npm），在命令提示符窗口执行命令node -v和npm -v查看是否安装成功。

    2. （可选步骤）在命令提示符窗口执行命令npm install -g cnpm --registry=https://registry.npm.taobao.org，该步骤目的为将镜像源更换为淘宝镜像源以加快下载速度。使用命令cnpm -v看是否安装成功。

    3. 在命令提示符窗口执行命令cnpm install hexo-deployer-git --save安装Hexo博客部署插件。

    安装Hexo博客框架

    1. 使用cnpm安装：cnpm install -g hexo-cli
    2. 查看是否安装成功：hexo -v

    初始化博客

    1. 在适当位置建立MyBlog文件夹用来存放博客相关文件。（文件夹名称任意）
    2. 切换至步骤1中创建的文件夹目录，执行命令hexo init初始化博客。
    3. 执行命令hexo s在本地启动博客，可以通过浏览器访问http://localhost:4000/进行预览。

    将博客部署到GitHub Pages

    1. 注册GitHub账号。（假设用户名为username）

    2. 创建一个仓库，仓库名必须设置为username.github.io，且属性需设置为Public。

    3. 修改Hexo博客配置文件：打开MyBlog/_config.yml，找到# Deployment，修改type的值修改为git，并新增字段repo: 'github仓库地址'和branch: master。（注意冒号后的空格不能省略）

    4. 若从未使用过Git，则需要先给git配置身份。

    5. 在命令提示符窗口执行命令hexo d将博客推送至GitHub仓库。（此步骤可能需要输入GitHub账号及密码）

    至此Hexo博客已经搭建成功，可以通过username.github.io进行访问

    更改主题及个性化设置

    1. 访问Hexo官网查找自己喜欢的主题。

    2. 进入主题的GitHub仓库页面，点击Code->Download ZIP下载对应的压缩包。

    3. 将压缩包解压后放入MyBlog/themes文件夹下。

    4. 修改Hexo博客配置文件：打开MyBlog/_config.yml，找到# Extensions，修改theme的值修改为对应主题的文件夹名称。
